Featured Projects

Tokyo Scene

     

TokyoScene is a Ruby on Rails web application that is optimized for mobile use and aims to gather and suggest music events happening in Tokyo. By using real-time data from Spotify, it creates event recommendations tailored to users' music tastes, providing a personalized list of upcoming shows.

Check it out!

USpeak4Me

     

USpeak4Me is a web application developed using Ruby on Rails that connects foreign residents in Japan with interpreters for translating between Japanese and English (or other languages). It assists with essential administrative tasks, such as visiting city halls, renting apartments, and making phone calls, helping to bridge communication gaps and making it easier for users to navigate local services.
